# Fardeen Ansari Portfolio

React/Vite portfolio website with dark/light theme, project previews, CV downloads, certificate previews, and a contact form that opens WhatsApp or email with the visitor's message prefilled.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The WhatsApp and email contact actions are configured in `src/App.jsx`.
